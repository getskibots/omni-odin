 How do I add a new language to the bot? How do I add a new language to the bot? Introduction Add a new language Introduction More languages - better customer experience. Follow these simple steps to make your chatbot a rrreal polyglot! 🌍 Add a new language Navigate to the Settings page → Languages (English language is set by default). In order to add additional language you should connect bot to DialogFlow or WitAi first (go to Integrations sub-tab of the Bot Settings tab). Click on '+ Add Language' button The following popup will appear: In the popup, choose the Language from the drop-down list and click "Add" button: New language is added to the Enterprise Platform and displayed on the list: Now you can add new language versions to your chatbot flow: You can also add phrases on added language to the FAQs and Intents in the NLP tab 

 ☝🏽Note! Built-in intent recognition service supports only English language. In case there is a DialogFlow added to the chatbot, some languages may not be supported. To add unsupported languages to the chatbot, connect the bot instance to WitAi. This can be done in ‘Bot Settings’→ ‘Integrations’.
