Prepare beforehand Add an FAQ Edit an FAQ Related articles Prepare beforehand Before adding an FAQ, we suggest that you prepare the following: a list of questions you are going to cover; a list of phrases: different formulations of the same question that a chatbot user may ask (e.g. "When do you work?", "Your working hours," "Tell me the working hours"); an answer to every question (create a separate atom for each question answer in the Flows tab). Add an FAQ In the Enterprise Platform, open the NLP tab → FAQ. Click the "Add new FAQ" button. ' Add new faq' button in the FAQ creation in the Enterprise Platform, NLP tab In the popup window, enter the FAQ name (it can be any name that would characterize the group of user questions). Add user phrases by pressing 'Enter' after each new phrase. These can be any questions or phrases the user might ask the chatbot. (e.g. "When do you work?", "Your working hours," "Tell me the working hours"). Select the chatbot reply in the 'Answers' drop-down menu. Press 'Create' button to finish. The newly added FAQ will appear on the list of FAQs. Edit an FAQ Find the required FAQ using the search. Press the "Edit" button in the FAQ's list. Edit input fields in the popup window: FAQ name, Questions, Answers (answers can be both edited and deleted). Press the 'Save' button. Related articles 'NLP' tab What is NLP Dialogflow integration
