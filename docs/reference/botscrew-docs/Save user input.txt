 Save user input Save user input Overview 
 Save user input is an atom builder element that provides the ability to set validation for the chatbot's user input and save user input as a value for the chatbot attribute. This allows to re-use it in the Flows tab on the Enterprise Platform so that users will be guided through different flows based on their input. Admin is able to select validation types “Email”, “Date”, “Time”, “Phone”, “Number”, and “Custom validation” from the dropdown “Validation”. In case of input was valid chatbot user will receive an atom set in "Valid input redirect". In case of input was invalid chatbot user will receive an atom set in "Invalid input redirect". 
 Validation types 
 In case the validation type is “Number” the combination of the digits is considered valid. Only integers are accepted. In case the validation type is “Phone” format “+12345678910” is considered valid. Country code is required. In case the validation type is “Date”, “Time” formats from the library are considered valid. The date entered by the user is then displayed in the attribute in the format yyyy-mm-dd. Time entered by the user is then displayed in attribute in format hh:mm. In case the validation type is “Custom validation” input that satisfies provided regex is considered valid.
