 How to create multiple welcome messages? How to create multiple welcome messages? 
 Introduction In this section, we will learn how to create multiple welcome messages within an atom with the "Random redirect" object. Step 1: Create different welcome message atoms. Select the "Flows" tab on the left-side panel. Click on the "Add a new atom" button in the "Basic chatbot configurations" flow. 3. Enter the atom name. Tap the "Enter" button on your keyboard. 4. Select the "Welcome message 1" atom to open the atom builder. 5. Click on the "Text" object on the bottom panel. 6. Enter a text to the atom. 7. Click on the "Attachment" button on the bottom panel. Select a gif file on your PC > upload the gif. 8. Click on the "Save" button. Step 2: Add the "Random redirect" object to the "Welcome message" atom. Select the "Welcome message" default atom. 3. Hover the mouse on the default "Text" object in the atom. 4. Click on the "Remove" button on it. 5. Click on the "Ellipsis" button on the bottom panel. 6. Select the "Random redirect" object on the pop-up. Step 3: Add the welcome message atoms into the "Random redirect" object of the "Welcome message" default atom. Click on the "Random atoms" field. 2. Select the "Test atom" atom. 3. Click on the "Save" button. Step 4: Test the random representation of the welcome message atoms in the widget. Click on the "Test widget" button on the top panel. 2. Open the widget by clicking on the bubble button. 3. Clear the cookie files on your browser. 4. Go back to the Enterprise Platform. Click on the "Test widget" button. 5. If you see that some changes are required, jump into Enterprise Platform → Flows tab to adjust your "Welcome message" atom. Congratulations! 🎉 You've just created a bot with multiple welcome messages! 👏 Users will never be bored while interacting with your chatbot!
