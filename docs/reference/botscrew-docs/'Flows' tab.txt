 'Flows' tab 'Flows' tab Introduction Flow Atom Atom editor How to build a flow Related articles Introduction Flows tab is THE place to draw up your chatbot's conversation map. Every user of the chatbot will be following the conversation based on the journey you build here. Let's learn about the main components of the Flows page to be able to build sophisticated flows 👇🏽 Flow Flow is a conversational line or dialog that guides the user to his desired goal. Under the Flows, you can find all your already existing flows and create new ones. You can search for a required flow using the search bar. 

 Feedback, Human handover, and Basic Chatbot Configurations flow are already provided by default. They are basic for all bots, but you can customize them as you wish. 
 Atom Atom is a piece of content, that is a part of the flow. It provides users with different types of information and helps to navigate through the flow to achieve the user's goal. Atom editor Atom editor is a toolbar on the left-hand side of the Flows page. With the editor, you can edit atoms, fill them in with the content, add redirect buttons, attachments or delete the added content.
